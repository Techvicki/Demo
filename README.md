# Demo
FogShield: Intelligent Safety Lighting &amp; Accident Detection

1️⃣ Project Overview

FogShield is a smart road safety system designed to reduce accidents caused by low visibility during foggy conditions, especially in remote or rural areas.
The system uses sensors and an Arduino microcontroller to detect fog, monitor movement, and identify potential accidents. When dangerous conditions occur, it automatically activates warning lights and alarms and sends emergency alerts through a GSM module.
This project aims to improve pedestrian and vehicle safety by creating an automated fog warning and accident alert system for highways, bridges, and mountain roads.

2️⃣ Problem Statement

Fog significantly reduces road visibility, which often leads to:
Vehicle collisions
Pedestrian accidents
Delayed emergency response in remote areas
Lack of real-time warning systems on highways
Most roads currently do not have automated systems to detect fog conditions or notify authorities when accidents occur, making rescue and prevention difficult.

3️⃣ Proposed Solution

FogShield provides an intelligent roadside safety system that continuously monitors environmental conditions and road activity.
The system works by:
Detecting fog density using a fog sensor
Activating high-visibility fog warning lights when visibility drops
Monitoring vehicle and pedestrian movement using ultrasonic and PIR sensors
Identifying possible accidents when movement stops unexpectedly
Sending SMS alerts via GSM module to emergency contacts or authorities
This ensures early warning and faster response, helping prevent accidents and save lives.
